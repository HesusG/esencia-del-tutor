# La esencia del tutor

Pieza visual e interactiva (scrollytelling) sobre lo que de verdad sostiene el acompañamiento tutorial.
Complemento del trabajo *Perfil del tutor* (Temas Selectos de Educación, UPAEP).

🔗 **Sitio:** https://hesusg.github.io/esencia-del-tutor/

Hecho con scroll suave (Lenis) y animaciones de scroll (GSAP ScrollTrigger): hero cinematográfico,
una constelación SVG interactiva de las seis esencias, parallax y reveals. Imágenes de Unsplash
(Vitaly Gariev, Fujiphilm). Respeta `prefers-reduced-motion`.
